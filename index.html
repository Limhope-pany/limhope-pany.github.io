套娃WP
压缩包分析、文档隐写、RC4解码
解题步骤：
1.开启环境后有一个附件（套娃.xlsx），将后缀.xlsx改为.zip，解压得到两个文件

·RC4data.txt（存放 RC4 密文U2FsdGVkX19uI2lzmxYrQ9mc16y7la7qc7VTS8gLaUKa49gzXPclxRXVsRJxWz/p）
·swpu.xlsx
所以处理swpu.xlsx
·什么是xlsx文件？
xlsx 是 Microsoft Excel 2007 及之后版本 默认的电子表格文件格式，全称 Office Open XML Spreadsheet。
（1）旧版 Excel（2003 及更早）后缀是 .xls；
（2）xlsx 是压缩包格式，底层本质是 ZIP 压缩包，内部存放 XML 描述文件、图片、嵌入附件、隐藏数据等
2.将swpu.xlsx后缀改为.zip，解压得到

·easyrc4.xlsx
·RC4key.zip（带密码，内部是存放密钥的RC4key.txt）
3.用010 Editor打开easyrc4.xlsx，导出为十六进制，搜索password

得到密码字符串：6e4c834d77cba03af41e1562a5bce84e
4.使用该密码解压RC4key.zip，打开RC4key.txt得到 RC4 密钥：ABCDEFGHIJKLMNOPQRSTUVWXYZ

·什么是RC4密钥？
RC4 是一种对称流加密算法：加密、解密用同一个密钥，加密和解密操作完全一样（异或可逆）。
明文 \oplus 密钥流 = 密文
密文 \oplus 密钥流 = 明文
所以 RC4加密 = RC4解密，运算无区别。
·如何解密：
（1）使用RC4命令工具,把加密后的密文保存为cipher.bin，在bash中输入：
rc4 “ABCDEFGHIJKLMNOPQRSTUVWXYZ” cipher.bin > plain.txt
打开plain.txt就是解密后的明文
（2）python脚本解密（最常用）
5.取出RC4data.txt内的密文，使用 RC4 解密工具（CyberChef），密钥填入ABCDEFGHIJKLMNOPQRSTUVWXYZ，解密得到：ef1a73d40977a49b99b871980f355757
6.flag为：NSSCTF{ef1a73d40977a49b99b871980f355757}

（2）reboot重启openEuler

（4）exit退出用户登录

（6）cd / ls 进入系统根目录/,列出所有一级目录

（7）cd ~ 和ls -a进入用户家目录，显示隐藏文件

（8）mkdir -p:在/tmp下创建多级目录结构Linux/pratice/test

（9）touch:切换到test目录，新建空文件note.txt

（10）cp：复制note.txt到上级practice目录，重命名为bak.txt

（11）mv：移动bak.txt到/tmp目录

（14）cat：完整查看文件内容

（15）less：分页查看

（16）find：在/etc目录下查找所有后缀为.conf的文件

（19）user01：创建用户user01，附属组加入study

Linux基础学习

1.安装openEuler，并将其导入虚拟机（有VM可以直接在VM中导入，没有的话可以下载VirtualBox）

在VM中导入流程如下：
步骤一：创建虚拟机
(1)安装openEuler:https://repo.openeuler.org/openEuler-20.03-LTS/ISO/x86_64/openEuler-20.03-LTS-x86_64-dvd.iso(保存在本地，不要放在中文路径)
(2)打开VM，在左上角点击【文件——新建虚拟机】

(3)配置类型选择【自定义(高级)——下一步】

(4)硬件兼容性：默认workstation 17.x——下一步
(5)安装来源：稍后安装操作系统(不要直接挂载ISO)——下一步

(6)客户机操作系统：单选【其他】；版本【其他64位】——下一步
(7)虚拟机名称与位置：名称写openEuler，存储路径最好不要放在C盘且路径不要包含中文——下一步
(8)处理器：2核2线程——下一步

(9)内存：4096MB——下一步

(10)网络类型：NAT——下一步
(11)I/O控制器：默认——下一步
(12)磁盘：选择【创建新虚拟磁盘】——下一步
(13)磁盘大小：50GB，选择【将虚拟磁盘存储为单个文件】——下一步
(14)磁盘文件名称默认——下一步——点击完成创建虚拟机

步骤二：挂载openEuler ISO 镜像
(1)在VM左侧选中新建好的虚拟机——点击【编辑虚拟机设置】
(2)选择【CD/DVD(SATA)】
(3)右侧选择【使用ISO映像文件】——【浏览】选中下载好的openEuler镜像
(4)点击【确定】保存
(5)点击【开启此虚拟机】
(6)选择【Install openEuler】
(7)按照向导设置语音、磁盘分区、root密码、创建普通用户，等待安装完成
·安装目标：选50GB SATA 磁盘——自动分区——点击【完成】
·网络和主机名：打开网卡开关（ens33），自动获取DHCP地址——自定义主机名——【完成】
·软件选择：左侧勾选【最小安装】——右侧勾选【开发工具】【无图形终端系统管理工具】——【完成】
·返回安装信息摘要界面，点击【开始安装】
·设置root密码并创建用户（勾选【将用户设为管理员】）
(8)等待安装完后点击【重启】
(9)重启后首次登录使用设置的root账号或者新建普通用户登录
(10)登录成功会出现 [root@openEuler-vm ~]# 命令提示符，输入 Linux 命令操作系统
